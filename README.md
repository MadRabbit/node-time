# NodeJS Response Times

A just a super simple hello world project in node.js to show that a NodeJS app
can respond in a sub-millisecond time easily.

![](./response-times.png)

```
Processor 1.2 GHz Intel Core M
Memory 8 GB 1600 MHz DDR3
```
